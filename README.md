- Python 3.x
- Django 2.x
- MySQL 5.x
- Vue.js

1. 安装Python环境
2. 安装Django框架
3. 安装MySQL数据库
4. 初始化数据库
5. 安装前端依赖
6. 启动后端服务
7. 运行前端项目
